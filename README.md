# JQuery Day 4 Warm-up!


## Instructions:

### 1. Selector Drills

Given the following HTML page:
```html
<!DOCTYPE html>
<html>
  <head>
    <script src="https://code.jquery.com/jquery-git.js"></script>
    <title>Fruits</title>
  </head>
  <body>
    <h1>Fruits</h1>    
    <h2 class="div">Orange</h2>
    <p id="h2">Banana</p>
    <div>Cherry</div>
  </body>
</html>
```
What would be the output of the following jQuery commands?

```javascript
$('h1').text();
```

```javascript
$('h2').text();
```

```javascript
$('#h2').text();
```

```javascript
$('div').text();
```

```javascript
$('.div').text();
```

```javascript
$('#div').text();
```
